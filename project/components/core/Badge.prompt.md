**Badge** — tiny count or status marker. Use for unread counts and short status labels.

```jsx
<Badge tone="brand">3</Badge>
<Badge tone="live">LIVE</Badge>
<Badge tone="neutral" size="sm">Draft</Badge>
<Badge tone="danger" dot />
```

Tones map to the semantic palette. `dot` renders a bare status dot. Keep content to a number or one short word.
